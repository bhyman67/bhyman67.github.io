PDF Table Parsing

As an avid Colorado outdoorsman, I wanted to pull some data from the deer statistics page on the CPW web site. I wrote a scraper to pull population estimates from 2010 to 2019. And then I plotted that data over time for each data analysis unit (DAU). The data was pulled from those PDFs via a tool called tabula-py

Algo Trading

Algorithmic trading is all about automating trades on the stock market. It’s a computerized trading system that can use variables such as price, timing, and volume in order to automatically make trades. And the more exciting piece is that you can develop logic using ML. Trade decisions that are both automated and computer informed… The fun part is in building that bot…

Currently I’m in the early stage of this field. My scripts are pretty trivial right now. And my focus is on basic analysis. But in due time, I’m hoping to approach expert status on this exciting topic.
